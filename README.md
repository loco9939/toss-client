# vite-settings

Vite + Vitest 세팅

## 설치

```bash
yarn install
```

## 실행

```bash
yarn dev
```

## 라이브러리 목록

### Dependency

- React v18
- React Router v6
- Styled-components v6
- Zustand

### Dev Dependency

- Vite v5
- Vitest v1
- TypeScript v5
- Prettier v3
- Eslint v8
- MSW v2
- React-Testing-Library v16
- Playwright v1
- Codeceptjs v3
- Dotenv v16
